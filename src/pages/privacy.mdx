# Privacy & Cookies
This website is an open knowledge portal published on GitHub Pages.

## What data we collect
We do not require account registration and we do not intentionally collect personal data through forms on the site.
If you contact us by email, we will receive the information you provide in your message.

## Analytics cookies
We use **Google Analytics only** if you explicitly accept analytics cookies in the consent banner.

If you accept analytics cookies, Google Analytics may collect information such as:
- pages visited,
- approximate location,
- browser and device information,
- referrer and interaction data.

We use this information only to understand which content is useful and improve the site.

## Hosting
The site is hosted on GitHub Pages. GitHub may process technical request data required to deliver the website, such as IP address, user agent, and server logs.

## Manage your cookie choice
You can reopen the cookie settings at any time by clicking the button below.

<button
  className="button button--primary"
  onClick={() => {
    if (typeof window !== 'undefined') {
      window.dispatchEvent(new Event('sfdevops:open-cookie-settings'));
    }
  }}>
  Open cookie settings
</button>

If you previously accepted analytics cookies and want to stop analytics, reject cookies in the banner and clear cookies for this site in your browser.
